# React Component Library

Welcome to our React Component Library! This collection showcases two powerful, accessible, and customizable components built with modern React patterns.

## 🚀 Components Overview

### InputField
A flexible input component that handles various states, variants, and user interactions.

**Key Features:**
- **Multiple Variants**: Filled, outlined, and ghost styles
- **Size Options**: Small, medium, and large
- **Validation States**: Error handling with clear messaging
- **Interactive Features**: Clear button and password toggle
- **Accessibility**: Full ARIA support and keyboard navigation
- **TypeScript**: Complete type safety

### DataTable
A comprehensive data table with sorting, selection, and state management.

**Key Features:**
- **Generic TypeScript**: Works with any data structure
- **Column Sorting**: Ascending, descending, and neutral states
- **Row Selection**: Single or multiple selection with callbacks
- **Loading & Empty States**: Graceful handling of different data states
- **Custom Rendering**: Flexible cell content with render functions
- **Accessibility**: ARIA compliant with keyboard navigation

## 🎯 Usage Examples

### Basic InputField
\`\`\`tsx
<InputField
  label="Email Address"
  placeholder="Enter your email"
  helperText="We'll never share your email"
/>
\`\`\`

### Advanced InputField
\`\`\`tsx
<InputField
  label="Password"
  type="password"
  showPasswordToggle
  showClearButton
  invalid={hasError}
  errorMessage="Password is required"
/>
\`\`\`

### Basic DataTable
\`\`\`tsx
<DataTable
  data={users}
  columns={[
    { key: 'name', header: 'Name', sortable: true },
    { key: 'email', header: 'Email', sortable: true }
  ]}
/>
\`\`\`

### Advanced DataTable
\`\`\`tsx
<DataTable
  data={products}
  columns={columnsWithCustomRendering}
  selectable
  onRowSelect={(rows) => handleSelection(rows)}
/>
\`\`\`
